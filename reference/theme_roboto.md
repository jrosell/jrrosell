# Sets a minimal theme using the Roboto font family

It requires roboto fonts installed in your O.S. and run z

## Usage

``` r
theme_roboto(
  base_size = 13,
  strip_text_size = 14,
  strip_text_margin = 6,
  subtitle_size = 14,
  subtitle_margin = 10,
  plot_title_size = 18,
  plot_title_margin = 12,
  ...
)
```

## Arguments

- base_size:

  = 11

- strip_text_size:

  = 12

- strip_text_margin:

  = 5

- subtitle_size:

  = 13

- subtitle_margin:

  = 10

- plot_title_size:

  = 16

- plot_title_margin:

  = 10

- ...:

  Other parameters passed to theme_set
