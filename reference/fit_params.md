# Fit a workflow with specific parameters

Fit a workflow with specific parameters

## Usage

``` r
fit_results(wf, resamples, param_info = NULL, grid = 10, fn = "tune_grid", ...)
```

## Arguments

- wf:

  workflow

- resamples:

  rset

- param_info:

  for tune\_\* functions

- grid:

  for tune\_\* functions

- fn:

  the name of the function to run when tuning

- ...:

  Optional engine arguments
